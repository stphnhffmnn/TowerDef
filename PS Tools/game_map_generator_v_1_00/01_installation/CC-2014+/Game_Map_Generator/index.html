HELP

  1. To create a new Game Map, just enter the map size and click CREATE.
  2. To generate a new element, create a basic shape first: pixel layer, vector shape, Smart Object or text layer are possible. To draw a pixel layer, simply click ADD LAYER. Now the element is selectable and can be clicked. Be aware that your basic shape is selected.
  3. To generate a new texture layer, first choose one of the textures in the panel. Then draw the shape into the mask of the new texture layer. Please note: areas in a mask that are marked black will be hidden, white areas are visible. intermediate grey coloured areas will be semi-transparent.
  4. Click on ICONS to use the basic icons to design your map. For always getting the same size of an icon use the ICON SIZE scaling. Draw the controller to a size between 1 – 100% and then click on the desired icon in the panel. You can also upload your own icons into the panel. Therefore choose an icon folder. Suitable for upload are PNG files and PSD files with only one layer.
  5. Please click to uninstall the Game Map Generator
